# منصة المقاول (El Moquwal) 🏗️

المنصة الأولى للمقاولات في مصر. تجمع منصة "المقاول" بين أصحاب العقارات الباحثين عن خدمات تشطيب أو بناء، وبين نخبة من المهندسين والمقاولين المعتمدين في مكان واحد لضمان جودة لا تُضاهى.

## 🛠️ المتطلبات الأساسية (Prerequisites)
تأكد من تثبيت البرامج التالية على جهازك قبل البدء:
- **Node.js**: إصدار 18 أو أحدث (موصى باستخدام الإصدار 20+ أو 22+).
- **MongoDB**: قاعدة بيانات (سواء مثبتة محلياً أو باستخدام MongoDB Atlas المرفوع على السحابة).
- **Git**: لاستنساخ المستودع (اختياري إذا قمت بتحميل الملف كـ ZIP).

## 🚀 كيفية تشغيل المشروع من الصفر

### 1. تثبيت المكتبات والاعتمادات (Dependencies)
كل الاعتمادات البرمجية (السيرفر وواجهات المستخدم) يتم إدارتها من خلال مجلد `backend`:

```bash
# الدخول إلى مجلد الباك-إند
cd backend

# تثبيت جميع المكتبات المطلوبة
npm install
```

### 3. إعداد بيانات الإدارة (Seeding Data) - [لأول مرة فقط]
لإنشاء مستخدم يحمل صلاحيات الأدمن (المدير) ليتمكن من إدارة المنصة:

```bash
# تأكد من أنك تعمل بداخل مجلد backend وتشغيل الأمر:
npm run seed:admin
```
*(تمت برمجة السكريبت ليقوم بضخ بيانات المدير الأساسية إلى قاعدة البيانات).*

### 4. تشغيل المشروع (Running the Server)
السيرفر الخاص بالباك-إند (Node.js/Express) يعمل كمضيف (Host) لملفات واجهة المستخدم (الفرونت-إند) برمجياً. لذلك، تشغيل السيرفر سيقوم بتشغيل المنصة بأكملها كـ Full-Stack.

للشروع في تشغيل المشروع، اكتب في الترمينال:
```bash
# تشغيل السيرفر في وضع المطور (يقرأ التحديثات تلقائيًا)
npm run dev

# أو في وضع الإنتاج
npm start
```

### 5. تصفح المنصة (Accessing the App)
الآن وقد نجح تشغيل السيرفر وظهرت رسالة أن التوصيل تم بنجاح، افتح متصفحك وادخل على الرابط التالي:
👉 **[http://localhost:4000](http://localhost:4000)**

## 🏗️ كيف تعمل هذه المنصة (بنية المشروع)؟
- **الواجهة الأمامية (Frontend):** 
مبنية باستخدام Vanilla JavaScript و HTML و المتغيرات المنسقة عبر CSS (لا تحتاج أدوات بناء معقدة مثل React أو Webpack)، ويتم تقديمها وربطها بشكل ديناميكي وثابت (Static Setup) من خلال السيرفر.
- **خلفية النظام (Backend):**
مبنية بـ Node.js و Express. وتتضمن:
  - إدارة وحماية التوجيه بالمصادقة (JWT Auth & Role Guards).
  - التقييد وحماية المسارات لمستخدمين بعينهم (صاحب مشروع، مقاول، أدمن).
  - الربط مع قاعدة بيانات MongoDB من خلال `Mongoose`.
  - معالجة وربط الأسعار بواسطة (Claude AI Estimator).
